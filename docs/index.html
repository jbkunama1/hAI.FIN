Features

Streamlit-Dashboard

Live-Signale, Portfolio-Simulation und Trade-Log auf Port 8501.

Quickstart

# 1. Klonen & konfigurieren
git clone https://github.com/jbkunama1/hAI.FIN.git
cd hAI.FIN && cp .env.example .env
joe .env   # Secrets eintragen

# 2. Lokal testen
pip install -r requirements.txt
python -m pytest tests/ -v
DRY_RUN=true python -m src.agent

# 3. Dashboard
bash dashboard/run.sh
# → http://localhost:8501

# 4. Docker Compose
docker compose up -d
# Agent + Dashboard laufen als Services
